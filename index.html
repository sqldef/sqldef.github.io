sqldef

sqldef is a CLI tool for diffing two SQL schemas. You can use it to manage the migration of RDBMSs using regular SQL DDLs.

Supported databases: MySQL, MariaDB, TiDB, PostgreSQL, SQL Server, and SQLite3.
